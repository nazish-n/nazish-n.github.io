# GitHub Pages Starter Site

This is a simple static personal website for GitHub Pages.

## How to publish

1. Create a GitHub repository named exactly:

   ```text
   YOUR-USERNAME.github.io
   ```

2. Upload these files to that repository:

   - `index.html`
   - `styles.css`
   - `assets/`

3. In GitHub, go to:

   ```text
   Settings → Pages
   ```

4. Under **Build and deployment**, select:

   ```text
   Source: Deploy from a branch
   Branch: main / root
   ```

5. Your site should appear at:

   ```text
   https://YOUR-USERNAME.github.io
   ```

## What to edit

- Replace `your.email@example.com` with your real email.
- Replace placeholder links for Google Scholar, GitHub, LinkedIn, papers, and code.
- Add your CV at `assets/Waleed_Akbar_CV.pdf`, or update the CV link in `index.html`.
- Replace the initials card with a photo if you want.
